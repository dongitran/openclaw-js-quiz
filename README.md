## Topics Covered

### JavaScript (Questions 1-50)
- Type coercion and equality
- Scope and hoisting
- Closures
- Event loop and asynchronous programming
- Promises and async/await
- `this` keyword
- Prototypal inheritance
- ES6+ features (Generators, Maps, WeakMaps)
- Shallow vs Deep copy
- Currying and functional programming
- Security (XSS, CORS)
- Performance optimization

### TypeScript (Questions 51-100)
- Interfaces vs Types
- Generics and Constraints
- Union and Intersection types
- Type Guards and Narrowing
- Utility Types (Partial, Pick, Omit, etc.)
- Mapped Types and Conditional Types
- Decorators
- strict mode and compiler options
- Discriminated Unions
- Template Literal Types
- infer keyword and type inference
- Declaration merging

## How to Use

1. Visit the [live site](https://dongitran.github.io/openclaw-mino-js-quiz/)
